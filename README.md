# Sustainable Product System
## Notes
Simple project for OOD course

## Description

A simple console application to manage products and materials and calculate environmental impact.

## Purpose

The system helps users understand products and how to recycle them.

## Functional Requirements

* Create a product
* List all products
* View product details
* Add and manage materials
* Calculate environmental impact
* Show recycling information

---

## Non-Functional Requirements

* Use layered structure
* Keep code simple and clear
* Make the system easy to test
* Separate logic from UI
